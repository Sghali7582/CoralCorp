Beneficios de ParLot

Interactividad

El diseño está planteado para  que el usuario este cómodo con las actividades.

Aprendizaje

Las lecciones están diseñadas para generar conocimiento de manera didáctica.

Compromiso Social

El objetivo de ParLot es ayudar a aquellas personas que sufren de algún problema de desarrollo social.

Usabilidad

La página es dinámica y muy fácil de usar para el usuario.

Intuitivo

La página esta diseñada de forma agradable para hacer sentir bien al usario en todo momento.

Sin fines de lucro

La idea de este proyecto surgió con el proposito de ayudar a las personas a comprender o manejar sus emociones, esto sin buscar un beneficio propio.
