Recursos digitales

    ¿Cuál elegir?



    LUMOSITY


    Ofrece un entrenamiento cognitivo diario con juegos que se basan en tareas neurocognitivas y neuropsicológicas comunes: velocidad, atención, memoria, flexibilidad y resolución de problemas, entre otras.



    AbaPlanet Lite


    Este se puede usar Es una app de aprendizaje para iPad que permite trabajar el reconocimiento de objetos y vocabulario básico a través de un sistema de inteligencia artificial pre-programado, que toma decisiones en base al rendimiento del niño, disminuyendo o acelerando el ritmo de aprendizaje.



    Soy Visual


    Es un sistema de comunicación aumentativa que incluye fotografías, láminas y diversos materiales gráficos, además de una app con ejercicios prácticos. Todos los recursos se ofrecen gratuitamente en la plataforma web www.soyvisual.org

    PARLOT

    Es un proyecto para ayudar a las personas con algún tipo de problema de desarrollo emocional, a través de lecciones que le apoyarán para desenvolverse de una mejor manera en el ambiente social en base a un cierto tipo de aprendizaje adaptativo.
